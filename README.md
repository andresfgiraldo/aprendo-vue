# aprendo-vue


### Instalar VueJS
```
sudo npm install -g @vue/cli
```


### Verificar la version de Vue
```
vue --version
```


### Inicializar el proyecto VUE
```
sudo vue create frontend-vuejs
```


### Ejecutar el proyecto
```
npm run serve
```
